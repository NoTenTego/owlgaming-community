# Contributing

Dzięki za zainteresowanie projektem. Każda pomoc jest mile widziana.

## Jak zgłosić błąd

Użyj zakładki [Issues](https://github.com/NoTenTego/owlgaming-community/issues) i wybierz szablon **Bug report**.
Podaj jak najwięcej szczegółów — wersję, kroki do odtworzenia, logi.

## Jak zaproponować zmianę

1. Sprawdź czy podobny issue już nie istnieje
2. Otwórz nowy issue z szablonem **Feature request**
3. Poczekaj na feedback zanim zaczniesz kodować

## Jak wysłać Pull Request

1. Zrób fork repozytorium
2. Utwórz gałąź od `main`:
   ```bash
   git checkout -b fix/nazwa-bledu
   ```
3. Wprowadź zmiany
4. Przetestuj lokalnie (patrz [README](README.md) – Opcja 1 lub 2)
5. Commituj z czytelnym opisem:
   ```bash
   git commit -m "fix: napraw błąd X w systemie Y"
   ```
6. Wypchnij gałąź i otwórz Pull Request na `main`

Każdy PR wymaga akceptacji maintainera przed mergem.

## Konwencje

- Trzymaj się [coding_conventions.md](coding_conventions.md)
- Jeden PR = jedna zmiana — nie łącz niezwiązanych poprawek
- Komentarze i opisy commitów po angielsku
